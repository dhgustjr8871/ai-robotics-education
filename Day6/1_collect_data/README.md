# Lecture 1: Pick and Place - 데이터셋 수집 

로봇이 사물을 인식하기 위해 YOLO와 같은 비전AI 를 이용합니다. 그러나 우리가 실습에서 사용하는 물건이나, 실무에서 사용하게 될 물건들은 기존의 YOLO를 학습시키기 위한 데이터셋에 포함이 되어있지 않을 가능성이 높습니다. 따라서 우리는 감지 대상이 될 물체에 대한 커스텀 데이터셋을 만들어야 합니다.
